## Development

### Monitor `src` directory and automatically build project

```bash
npm run start:dev
```

### Install patchage from source

```bash
git clone ...
cd patchage
npm install
npm run build
npm install -g .
```

The binary `patchage` is now available:

```bash
patchage
```
